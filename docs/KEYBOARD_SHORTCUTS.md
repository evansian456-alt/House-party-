# Keyboard Shortcuts Reference

**Quick access to DJ mode controls without touching your screen**

This document provides a complete reference of all available keyboard shortcuts in Phone Party.

---

## 🎹 DJ Mode Shortcuts

The following keyboard shortcuts are available when you're in DJ mode (hosting a party):

### Playback Controls

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Space` | Play/Pause | Toggle playback of the current track |
| `N` | Next Track | Skip to the next track in the queue |
| `M` | Mute/Unmute | Toggle audio mute (visual feedback shown) |

### Navigation & Queue

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Q` | Queue Track | Open the "Add Track" dialog to queue a new song |
| `Esc` | End Party | Exit DJ mode and return to landing (with confirmation) |

---

## 📝 Usage Notes

### When Shortcuts Are Active

- Keyboard shortcuts **only work in DJ mode** (when you're the party host)
- Shortcuts **do not work** when typing in text fields (message input, track name, etc.)
- Shortcuts **do not work** on the landing page or guest view

### Visual Feedback

- **Mute Toggle**: Shows a toast notification ("🔇 Muted" or "🔊 Unmuted")
- **Play/Pause**: The play/pause button icon updates automatically
- **End Party**: Shows a browser confirmation dialog before exiting

### Browser Compatibility

- All modern browsers support these keyboard shortcuts
- No special permissions required
- Works on desktop and laptop computers
- Mobile devices typically don't have physical keyboards, so shortcuts are not applicable

---

## 🎯 Quick Tips

### Focus Management

Shortcuts are disabled when you're typing in an input field or textarea to avoid conflicts. Click outside the input field to re-enable shortcuts.

### Confirmation Dialogs

The **Esc** key shows a confirmation dialog before ending your party. This prevents accidental party termination.

### Accessibility

Keyboard shortcuts improve accessibility for:
- Users who prefer keyboard navigation
- Power users who want faster controls
- DJs managing high-energy parties where touching the screen is difficult
